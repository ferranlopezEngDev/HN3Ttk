# Examples

These scripts are practical tutorials. They are intended to be opened, read and
executed by someone who is learning the package for the first time.

Recommended order:

1. `01_single_pipe_step_by_step.py`
2. `02_parallel_pipes_step_by_step.py`
3. `03_three_reservoirs_step_by_step.py`
4. `04_compare_solvers.py`
5. `05_export_results.py`
6. `06_medium_generic_network.py`

Run them from the repository root, for example:

```bash
python examples/01_single_pipe_step_by_step.py
```

All examples are self-contained and do not require external data files.

Plot-oriented exploratory scripts live in [examples/plots](/home/chapo518/DEVcode/TFG/HN3Ttk/examples/plots/README.md).
