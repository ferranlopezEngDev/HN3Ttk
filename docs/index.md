# HN3Ttk Documentation

HN3Ttk is a hydraulic network toolkit focused on steady-state modelling,
nonlinear residual assembly, dense Jacobian assembly, solver experimentation
and reproducible academic examples.

It is especially useful for:

- learning how hydraulic network equations are assembled
- comparing Newton-based solvers
- building small and medium teaching examples
- exporting solver results to JSON and CSV

## Project Status

HN3Ttk 0.1.0 is an academic prototype. It is intended for learning,
prototyping and solver comparison.
